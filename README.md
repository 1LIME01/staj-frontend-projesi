# Kids Tree - Staj Projesi

Bu proje, staj kapsamında geliştirilen modern bir frontend uygulamasıdır.

## 🛠 Kullanılan Teknolojiler
* **Vue.js 3** (Composition API)
* **Vite** (Build Tool)
* **TypeScript**
* **Tailwind CSS** (Stil)

## 🚀 Kurulum Adımları
1. Projeyi bilgisayarınıza klonlayın:
   `git clone [[SENIN-REPO-LINKIN](https://github.com/1LIME01/staj-frontend-projesi)]`
2. Proje dizinine girin:
   `cd staj-frontend-projesi`
3. Gerekli kütüphaneleri yükleyin:
   `npm install`

## ▶️ Çalıştırma Komutları
* **Geliştirme sunucusunu başlatmak için:**
  `npm run dev`
* **Projeyi build etmek için:**
  `npm run build`
